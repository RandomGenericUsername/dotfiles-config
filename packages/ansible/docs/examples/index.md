# Examples

This section provides practical examples of using the ansible package.

## Available Examples

### [Basic Usage](basic-usage.md)

Common usage patterns and everyday commands for running the playbook.

### [Custom Paths](custom-paths.md)

Examples of overriding default paths and directories using variable overrides.

### [Selective Installation](selective-install.md)

Using tags to install specific software packages without running the entire playbook.

## Example Categories

### Quick Start Examples

For users who want to get started quickly with minimal configuration.

### Advanced Examples

For users who need custom configurations, path overrides, or integration with external systems.

### Testing Examples

For users who want to test changes before applying them.

## Running Examples

All examples assume you're in the ansible package directory:

```bash
cd /path/to/dotfiles/config/packages/ansible
```

Or that you're using absolute paths to the inventory and playbook files.
